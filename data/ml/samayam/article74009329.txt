ജമ്മു കശ്മീരിന് പ്രത്യേക പദവി ഉറപ്പ് നൽകുന്ന ആർട്ടിക്കിൾ 370 എടുത്ത് കളഞ്ഞാൽ ഭൂകന്പം വന്ന് കാശ്മീർ ഇന്ത്യയിൽ നിന്ന് വേർപെട്ട് പോകും എന്ന് ഒമർ അബ്ദുള്ള പറഞ്ഞു എന്നാണ് പ്രധാനമന്ത്രി നരേന്ദ്രമോദി ഫെബ്രുവരി ആറാം തീയതി ലോക്സഭയിൽ പ്രസംഗിച്ചത്. എന്താണ് ഇതിന്റെ സത്യാവസ്ഥ എന്ന് നോക്കാം. 

ആരാണീ ഒമർ അബ്ദുള്ള? ജമ്മു കശ്മീരിന്റെ മുൻ മുഖ്യമന്ത്രിയും നാഷണൽ കോൺഫറൻസ് പാർട്ടി നേതാവുമാണ് ഒമർ അബ്ദുള്ള. ശരിക്കും ഒമർ അബ്ദുള്ള അങ്ങനെ പറഞ്ഞോ. ഒമർ അബ്ദുള്ള പറഞ്ഞു എന്ന് പ്രധാനമന്ത്രി നരേന്ദ്ര മോദി പറഞ്ഞു. നരേന്ദ്ര മോദി എപ്പോൾ എവിടെ പറഞ്ഞു. ഫെബ്രുവരി ആറാം തീയതി നരേന്ദ്ര മോദി ലോക്സഭയിൽ പറഞ്ഞു. നരേന്ദ്രമോദിയുടെയും ബിജെപിയുടെയും യൂടൂബ് ചാനലുകളിൽ മോദിയുടെ ഈ പ്രസംഗം അപ്ലോഡ് ചെയ്തിട്ടുണ്ട്. ഇനി എന്താണ് പ്രധാനമന്ത്രി നരേന്ദ്രമോദി പറഞ്ഞതിലെ സത്യാവസ്ഥ എന്ന് നോക്കാം. 

 
 

ഒമർ അബ്ദുള്ള പറഞ്ഞതായി നരേന്ദ്രമോദി ആരോപിക്കുന്ന കാര്യങ്ങൾ ഗൂഗിളിൽ തിരഞ്ഞ് പോയാൽ 2014 മെയ് മാസത്തിൽ പ്രസിദ്ധീകരിച്ച ഒരു ആർട്ടിക്കിൾ കാണാം. ഈ ആർട്ടിക്കിളിൽ ഒമർ അബ്ദുള്ള, ആർട്ടിക്കിൾ 370 എടുത്ത് കളഞ്ഞാൽ ഭൂകന്പം വന്ന് കാശ്മീർ ഇന്ത്യയിൽ നിന്ന് വേർപെട്ട് പോകും എന്ന് പറയുന്നുണ്ട്. താൻ പറയുന്നത് ഒരു വലിയ രഹസ്യമാണ്. എൻഡിഎ സർക്കാർ ഇക്കാര്യം പരസ്യമാക്കാൻ തന്നെ നിർബദ്ധിതനാക്കിയിരിക്കുകയാണ്. തന്റെ പൂർവപിതാമഹന്മാർ കല്ലിൽ കൊത്തിവെച്ച കാര്യമാണ്. അന്ന് ആർട്ടിക്കിൾ നിലവിൽ വന്നിട്ട് പോലുമില്ല. നോക്കണേ, അവരുടെ ഒരു ദീർഘദൃഷ്ടി - ഒമറിനെ ഉദ്ധരിച്ച് ഫേക്കിങ് ന്യൂസ് എന്ന വെബ്സൈറ്റ് എഴുതുന്നു.

എന്താണ് ഇതിലെ പ്രശ്നമെന്ന് മനസിലായോ. ഫേക്കിങ് ന്യൂസ് എന്ന വെബ്സൈറ്റ് തന്നെയാണ് ഇവിടെ മോദിജിക്ക് പണി കൊടുത്തത്. സറ്റയർ ആർട്ടിക്കിളുകൾ പ്രസിദ്ധീകരിക്കുന്ന വെബ്സൈറ്റാണ് ഫേക്കിംഗ് ന്യൂസ്. ഇതിൽ പ്രസിദ്ധീകരിക്കുന്ന ആർട്ടിക്കിളുകളെല്ലാം ഭാവന മാത്രമാണ് എന്നും ഇത് വായിച്ചിട്ട് ആരും ഇതിൽ പറയുന്ന കാര്യങ്ങൾ സത്യമാണ് എന്ന് തെറ്റിദ്ധരിക്കരുത് എന്നും ഡിസ്ക്ലൈമർ കൊടുത്തിട്ടുള്ള വെബ്സൈറ്റാണ് ഇത്. ഫാക്ട് ചെക്കിംഗ് സെഷനുകളിൽ വ്യാജ വാർത്തകൾ പ്രസിദ്ധീകരിക്കുന്ന വെസ്ബൈറ്റുകളെക്കുറിച്ച് പറയുന്പോൾ ആദ്യം തന്നെ ഉദാഹരിക്കാറുള്ള ഒന്നാണ് ഫേക്കിംഗ് ന്യൂസ്.

<br>എന്താണ് ഇതിലെ പ്രശ്നമെന്ന് മനസിലായോ. ഫേക്കിങ് ന്യൂസ് എന്ന വെബ്സൈറ്റ് തന്നെയാണ് ഇവിടെ മോദിജിക്ക് പണി കൊടുത്തത്. സറ്റയർ ആർട്ടിക്കിളുകൾ പ്രസിദ്ധീകരിക്കുന്ന വെബ്സൈറ്റാണ് ഫേക്കിംഗ് ന്യൂസ്. ഇതിൽ പ്രസിദ്ധീകരിക്കുന്ന ആർട്ടിക്കിളുകളെല്ലാം ഭാവന മാത്രമാണ് എന്നും ഇത് വായിച്ചിട്ട് ആരും ഇതിൽ പറയുന്ന കാര്യങ്ങൾ സത്യമാണ് എന്ന് തെറ്റിദ്ധരിക്കരുത് എന്നും ഡിസ്ക്ലൈമർ കൊടുത്തിട്ടുള്ള വെബ്സൈറ്റാണ് ഇത്. ഫാക്ട് ചെക്കിംഗ് സെഷനുകളിൽ വ്യാജ വാർത്തകൾ പ്രസിദ്ധീകരിക്കുന്ന വെസ്ബൈറ്റുകളെക്കുറിച്ച് പറയുന്പോൾ ആദ്യം തന്നെ ഉദാഹരിക്കാറുള്ള ഒന്നാണ് ഫേക്കിംഗ് ന്യൂസ്.

	function getmatchupdatejsondata() {
		//console.log("==getmatchupdatejsondata==");
	   	 
	   	 $.ajax({
             method: "GET",
             url: "https://s3-ap-southeast-1.amazonaws.com/navbharattimes/webappcricwidget.htm"
         })
         .done(function(json) {
             json = json.Calendar;
             var livecoverage = false;
             
             var matchExist = false;
             var matchIndex = 0;
             var matchCounter = 0;
             
             $('.cricketContainerData').html('');
             
             for (i = 0; i < json.length; i++) {
                 if ((json[i].live == 1|| json[i].live == 3 || json[i].live == 4)) {
                    
                    $(".miniscorecard").show();
                    livecoverage = true;
                     
                    //console.log("seriesname : "+ json[i].seriesname.toLowerCase()); 
                     
                    if((json[i].seriesname.toLowerCase() == 'indian premier league') || (json[i].seriesname.toLowerCase() == 'icc cricket world cup') || (json[i].teama_short == 'IND') || (json[i].teamb_short == 'IND')){
                       matchExist=true;
                       matchIndex=i;
                       //console.log("matchIndex : "+ matchIndex);
                       scorewidgetdata(json[matchIndex]);
                       matchCounter++;
                    }
                 
                 }
             
                if (livecoverage == false) {
                    $(".miniscorecard").hide();
                    clearInterval(notCntTimer);
                }
                
             }//end for loop
            
            //console.log("matchCounter :" + matchCounter);
            if(matchCounter>1){
              $('.cricketSliderArrow').show();
            }
           
        });
     }
